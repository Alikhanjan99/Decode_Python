# Simple Multi Brute Force

Sebuah Script Untuk Mengkrek Akun Epep Emel Dan Pubji

# Cara Install

$ pkg update && upgrade  
$ termux-setup-storage  
$ pkg install python  
$ pkg install git  
$ pip install bs4  
$ pip install requests  
$ pip install mechanize  
$ pip install futures  
$ rm -rf smbf   
$ git clone https://github.com/Rian-YNTKTS/smbf

# Menjalankan Script

$ cd smbf   
$ python run.py
